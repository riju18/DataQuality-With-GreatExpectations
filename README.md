# DataQuality-With-GreatExpectations

# How to start

+ make a python venv
    ```sh
    python3 -m venv venv_name
    ```

+ activate the env
    + linux/mac
        ```sh
        source venv_name/bin/activate
        ```
+ install libraries
    ```sh
    pip3 install -r requirements.txt
    ```

+ clone the repo
+ run ```csv_data_validate.py```
